# Math-Quiz-Game (C++)

A console-based Math Quiz game written in C++ using fundamental programming concepts such as Structures, Enums, and Functions.

## Features
* Custom Questions Count: Allows the user to select between 1 to 10 questions per quiz.
* Question Difficulty Levels: Supports Easy, Medium, Hard, and a Mixed option where difficulty is randomized for each question.
* Arithmetic Operations: Includes Addition, Multiplication, Subtraction, Division, and a Mixed option that randomizes the operator per question.
* Interactive Screen Colors: Changes the console background color to Green for correct answers and Red (with an audio alert) for incorrect ones.
* Detailed Statistics: Displays a final performance dashboard showing total questions, right answers, wrong answers, and whether the user passed or failed.

## Requirements and Compilation
To run this game, you need a C++ compiler or an IDE like Visual Studio.

1. Open Math Quiz Game.cpp in your C++ IDE.
2. Compile and run the project.
